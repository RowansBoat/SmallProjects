### 3. **FTP Client**
   - **Description**: A command-line FTP client built with raw Python sockets. Supports file uploads, downloads, directory listing, and file management through passive mode connections—no ftplib dependency.
      - Note: May not work with all FTP servers, as it was originally built for a specific server.
      - ./ftpclient.py ls ftp://user:pass@host/path/          # List directory
      - ./ftpclient.py cp ftp://host/file.txt ./local.txt     # Download
      - ./ftpclient.py cp ./local.txt ftp://host/file.txt     # Upload
      - ./ftpclient.py rm ftp://host/file.txt                 # Delete file
      - ./ftpclient.py mkdir ftp://host/new_dir/              # Create directory
